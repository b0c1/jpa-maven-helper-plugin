jpahelper:generate-entities

Full name:

hu.javaportal.maven.plugin:jpa-maven-helper-plugin:1.0.0-SNAPSHOT:generate-entities

Description:

Collect Entity classes and extend the generated content

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: test.
  • Binds by default to the lifecycle phase: process-test-classes.

Required Parameters

Name Type Since Description
outputDirectory File - Output directory. Compiled entities.
Default value is: ${project.build.outputDirectory}.
persistenceXml File - Location of the persistence.xml.
Default value is: ${project.build.outputDirectory}${file.separator}META-INF${file.separator}persistence.xml.
testOutputDirectory File - Test files output directory. Compiled test entities.
Default value is: ${project.build.testOutputDirectory}.
testPersistenceXml File - Location of the test persistence.xml.
Default value is: ${project.build.testOutputDirectory}${file.separator}META-INF${file.separator}persistence.xml.

Optional Parameters

Name Type Since Description
filters UnitClassFilter[] - Unit Filters <filters> <filter> <name>Persistence unit name</name> <packages> <package>com.xxx</package> <package>com.yyy</package> </packages> </filter> </filters>
